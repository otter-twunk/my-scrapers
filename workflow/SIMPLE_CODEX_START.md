# Simple Codex Start

Use this minimal prompt after the scraper folder has been scaffolded and researched:

```text
Use the files in `scrapers/<site-folder>/` and build the scraper.

Start with:
- `SCRAPER_SPEC.json`
- `PERPLEXITY_TO_CODEX_HANDOFF.md`
- `CODEX_PROMPT.md`

Then implement the scraper in that folder and validate it against `workflow/NEW_SCRAPER_CHECKLIST.md`.
```

This assumes Perplexity has already chosen the folder name, structured the folder, and populated the handoff files.
